# CTK46 Lab01

Portfolio project built with Next.js App Router + Tailwind.

## Run

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Routes

- `/blog` — Blog list/detail (data fetching)
- `/guestbook` — Guestbook (API routes + Server Actions)
- `/pokemon` — PokéAPI assignment: list + detail (dynamic route)

## Build

```bash
npm run build
npm run start
```
